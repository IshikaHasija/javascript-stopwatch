⏱️ JavaScript Stopwatch

A simple Digital Stopwatch built using HTML, CSS, and JavaScript.
This project accurately tracks elapsed time using Date.now() and updates the display dynamically.

🚀 Features

▶️ Start the stopwatch
⏸️ Stop / Pause the timer
🔄 Reset the stopwatch
⏱️ Millisecond precision
🎯 Clean and simple UI

🛠️ Technologies Used
HTML – Structure of the stopwatch
CSS – Styling the UI
JavaScript – Timer logic and DOM manipulation

⚙️ How It Works
The stopwatch calculates time using JavaScript timestamps.
Date.now() gets the current time.
setInterval() updates the stopwatch every 10 milliseconds.
clearInterval() stops the timer when needed.
The elapsed time is then converted into:
Hours
Minutes
Seconds
Milliseconds
